# Graphene Cloud Landing Page

This repository contains the Graphene Cloud landing page

## About Graphene Cloud

Graphene Cloud is a privacy-first cloud storage platform. Files are encrypted on your device before upload to the server in a provider-blind architecture. We never receive your encryption keys, ever. Filenames are obfuscated, unique per-file keys are deterministically derived using quantum-hardened, hash-based derivation with 512-bit key material, and no party is treated as secure, along with many additional security features.
